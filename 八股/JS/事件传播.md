### 阶段

捕获 → 目标 → 冒泡

先执行捕获阶段，此时事件由根一直传播到目标，如果有设置 addEventListener(,,true) 监听，则执行

然后将目标元素的所有事件函数，不管第三个参数是 true 还是 false

最后执行冒泡阶段，由目标元素一直传播回根

### 具体

1. **捕获阶段（Capturing Phase）​**​  
   事件从最外层元素（如`window`或`document`）​**​ 向下传播 ​**​ 到目标元素。在此阶段，若父元素通过`addEventListener(event, handler, true)`注册了捕获监听器，会触发其处理函数

2. ​**​ 目标阶段（Target Phase）​**​  
   事件到达目标元素（触发事件的元素），执行目标元素上绑定的所有事件处理函数（无论第三个参数是否设置捕获）

3. ​**​ 冒泡阶段（Bubbling Phase）​**​  
   事件从目标元素 ​**​ 向上传播 ​**​ 回最外层元素。若父元素通过`addEventListener(event, handler, false)`（默认值）注册了冒泡监听器，会触发其处理函数

### 示例代码

```js
<div id="outer">
  <div id="inner">
    <button id="btn">点击我</button>
  </div>
</div>

<script>
  const outer = document.getElementById('outer');
  const inner = document.getElementById('inner');
  const btn = document.getElementById('btn');

  // 捕获阶段触发（第三个参数为true）
  outer.addEventListener('click', () => console.log('捕获阶段：outer'), true);
  inner.addEventListener('click', () => console.log('捕获阶段：inner'), true);

  // 目标阶段触发（默认冒泡阶段绑定）
  btn.addEventListener('click', () => console.log('目标阶段：btn'));

  // 冒泡阶段触发（第三个参数为false或省略）
  inner.addEventListener('click', () => console.log('冒泡阶段：inner'));
  outer.addEventListener('click', () => console.log('冒泡阶段：outer'));
</script>
```

执行结果：

1. `捕获阶段：outer`
2. `捕获阶段：inner`
3. `目标阶段：btn`
4. `冒泡阶段：inner`
5. `冒泡阶段：outer`
